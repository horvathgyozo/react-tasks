## First Steps

Installation and initial setup. For this, you will need an installed [Node.js](https://nodejs.org/en/) environment.

1. Create a React application using [Vite](https://vitejs.dev/guide/#scaffolding-your-first-vite-project). Use TypeScript.
2. After installation, start it (`npm run dev`).
3. Analyze the generated directory structure and the contents of the files.
4. Make simple changes to the files and observe their effects.
5. In the web developer toolbar, observe how the DOM updates when the page changes.
